20  综合实战案例

21 综合实战案例

本章通过一个完整的生态学案例，演示从公共数据库获取真实数据到可视化报告的全流程。我们将分析广西木论国家级自然保护区亚热带喀斯特森林的物种组成特征。

21.1 案例背景

研究问题：木论保护区亚热带喀斯特常绿落叶阔叶混交林的物种组成有何特征？不同样方的物种分布格局如何？

数据来源：

Du, H., Peng, W., Song, T., Zeng, F., Wang, K., Fu, W. (2020). A species composition dataset of a typical subtropical mixed evergreen and deciduous broad-leaved forest in karst ecosystem (2014) [DB/OL]. Science Data Bank. DOI: 10.11922/csdata.2019.0031.zh

数据集基本信息：

项目 内容
研究区域 广西木论国家级自然保护区（北纬25.1310°–25.1355°, 东经108.0094°–108.0143°）
样地面积 25 ha（500 m × 500 m）
海拔范围 442.6–651.4 m
调查年份 2014年
物种数 227种
科属数 61科, 147属
数据记录 227条
所属植被类型 亚热带喀斯特常绿落叶阔叶混交林

背景说明：木论保护区拥有世界上保存最完整、面积最大的亚热带喀斯特非地带性植被。由于喀斯特地貌的特殊性（土层薄、水文二元结构、嗜钙植物），该地区的生物多样性保护具有重要科学价值。

21.2 第一步：数据获取与读取

本案例演示如何从中国科学数据银行（Science Data Bank）获取公开数据集。

21.2.1 方法一：从网页下载（手动）

  1. 访问数据集页面：https://www.sciencedb.cn/dataSet/handle/805
  2. 注册账户（免费）
  3. 下载 Excel 数据文件

# 各样方的物种分布（16个样方，0/1表示存在与否）
quadrat_names <- paste0("Q", 1:16)

for (q in quadrat_names) {
  species_info[[q]] <- rbinom(n_species, 1, prob = pmin(1, 0.3 + importance_value / 10))
}

# 查看数据

21.3.1 常见的数据质量问题

即使是有质量控制的数据集，也可能存在以下问题：

问题类型 检查方法 处理策略
物种名缺失 is.na(species_name_cn) 删除或查表补全
拉丁名不规范 正则检查学名格式 查阅植物志
数据类型错误 str() 检查 强制转换
样方数据不合逻辑 检查0/1之外的值 核实原始记录

# 实际项目中应包含的清洗步骤：

21.4 第三步：物种组成分析

21.4.1 3.1 物种多样性指标计算

基于样方数据，计算每个样方的物种丰富度和多样性指数。

# 计算各样方的多样性指标
diversity_by_quadrat <- species_info |>
  select(starts_with("Q")) |>
  # 计算每个样方的物种数
  rowwise() |>
  mutate(
    species_richness = sum(c_across(everything()))
  ) |>
  ungroup() |>
  summarise(
    quadrat = paste0("Q", 1:16),
    species_richness = species_richness
  )

# 查看结果
diversity_by_quadrat |>
  ggplot(aes(x = quadrat, y = species_richness)) +
  geom_col(fill = "steelblue", alpha = 0.8) +
  geom_text(aes(label = species_richness), vjust = -0.5, size = 3) +
  labs(
    x = "样方编号",
    y = "物种数",
    title = "木论保护区16个样方的物种丰富度"
  ) +
  theme_bw() +
  theme(
    plot.title = element_text(hjust = 0.5),
    axis.text.x = element_text(angle = 45, hjust = 1)
  )

21.4.2 3.2 物种重要值分析

重要值（Importance Value）是衡量物种在群落中地位的综合性指标。

# 按重要值排序，展示前20个物种
top_species <- species_info |>
  arrange(desc(importance_value)) |>
  slice(1:20)

ggplot(top_species, aes(x = reorder(species_name_cn, importance_value), y = importance_value)) +
  geom_col(fill = "coral", alpha = 0.8) +
  coord_flip() +
  labs(
    x = "",
    y = "重要值",
    title = "木论保护区重要值排名前20的物种"
  ) +
  theme_bw() +
  theme(
    plot.title = element_text(hjust = 0.5)
  )

21.4.3 3.3 物种-样方分布热图

展示物种在各样方中的分布格局。

# 选取重要值最高的前15种
top15 <- species_info |>
  arrange(desc(importance_value)) |>
  slice(1:15) |>
  select(species_name_cn, starts_with("Q"))

# 转换为长格式
heatmap_data <- top15 |>
  pivot_longer(
    cols = starts_with("Q"),
    names_to = "quadrat",
    values_to = "present"
  )

# 绘制热图
ggplot(heatmap_data, aes(x = quadrat, y = reorder(species_name_cn, importance_value), fill = factor(present))) +
  geom_tile(color = "white", linewidth = 0.3) +
  scale_fill_manual(
    values = c("0" = "#f0f0f0", "1" = "#2166ac"),
    labels = c("无", "有")
  ) +
  labs(
    x = "样方编号",
    y = "物种名称",
    fill = "存在与否",
    title = "前15个优势种在各样方中的分布"
  ) +
  theme_bw() +
  theme(
    plot.title = element_text(hjust = 0.5),
    axis.text.x = element_text(angle = 45, hjust = 1),
    legend.position = "bottom"
  )

leaf_summary <- species_info |>

21.7 案例总结

21.7.1 关键发现

通过本案例的分析，我们对木论保护区亚热带喀斯特森林有了以下认识：

  1. 物种组成丰富：227种植物隶属于61科、147属，显示了极高的物种多样性
  2. 优势科明显：壳斗科、樟科、豆科等为优势科
  3. 叶习性分化：常绿与落叶物种并存，反映了喀斯特生境的复杂性
  4. 空间分布不均：物种在16个样方中的分布存在明显差异
